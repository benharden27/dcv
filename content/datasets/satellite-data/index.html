Satellite Data

There are a number of environmental satellites in orbit around earth gathering a range of information.

Below I’ve compiled well-formatted data sets for a few examples, but you can find many more online. There are many tools, but the following resources might be most helpful in your search for appropriate data (because they allow you to subset larger data sets by sub-region and then download a csv file):

  • NASA PODAAC Live Access Server (LAS)
  • NOAA ERDDAP server

Tip: You will probably need to use one of the geom_raster() functions combined with the mapping example to make a compelling map.

Sea Surface Temperature

Well formatted SST Data from GHRSST Level 4 MUR 0.25deg Global Foundation Sea Surface Temperature Analysis (v4.2) is here. I accessed this data from the LAS

The fields are:

  • dttm: Date and time
  • lon: Longitude (°E)
  • lat: Latitude (°N)
  • sst: Sea Surface Temperature (°C)

Sea Surface Salinity

Well-formatted SSS data from Aquarius Official Release Level 3 Sea Surface Salinity Standard Mapped Image Monthly Data V5.0 is available here. I accessed this data from the LAS

  • dttm: Date and time
  • lon: Longitude (°E)
  • lat: Latitude (°N)
  • sss: Sea Surface Salinity

Ocean Chlorophyll-a

Chlorophyll-a data for our region from Aqua MODIS, NPP, L3SMI, Global, 4km, Science Quality, 2003-present (8 Day Composite) is available here. I accessed this data from ERDDAP.

The original data is 4km resolution (~0.04 deg), but these files are big (127 MB) so I’ve subsampled the above data to be on the same resolution as the SST and SSS data (0.25 deg)

The fields are:

  • dttm: Date and time
  • lon: Longitude (°E)
  • lat: Latitude (°N)
  • chla: Chlorophyll-a concentration (mg m-3)

Ocean Currents

Ocean current data from OSCAR Third Degree Resolution Ocean Surface Currents, 1992-current YEARS OSCAR Third Degree Resolution Ocean Surface Currents is available here. This data is from the ERDDAP site.

The fields are:

  • dttm: Date and time
  • lon: Longitude (°E)
  • lat: Latitude (°N)
  • depth: Depth of modeled currents (m)
  • u: east-west component of current (m/s)
  • v: north-south component of current (m/s)
  • speed: total current speed (m/s)
